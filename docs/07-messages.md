# WhatsApp Messages

## Sending a Message
1. Open a lead's Edit page
2. Go to the **Messages** tab
3. Verify the customer phone number is displayed
4. Type your message in the text area
5. Click **Send Message**

**Note:** The lead must have a phone number for the WhatsApp option to appear.

## Message History
The right panel shows all sent messages with:
- Status badge (Sent / Failed)
- Message text
- Sender name
- Date and time

## Message Logs
Access: **Messages** in sidebar.

A centralized view of all WhatsApp messages sent across all leads.

### Columns
- Lead (linked to edit page)
- Phone number
- Message preview
- Status (colored badge)
- Sent By
- Date

### Access
- **Admin/Manager**: See all messages
- **Sales Officer**: See only messages from leads they own

## Restrictions on Closed Leads
When a lead is Closed Won or Closed Lost:
- **Non-admin users**: Cannot send messages — the send form is replaced with a lock notice
- **Administrator**: Can still send messages
