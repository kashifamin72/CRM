# User Profile

Access: Click your name/avatar in the top-right corner → **Profile**.

## Profile View
- **Hero Card**: Gradient background with large avatar (photo or initials), name, email, role badges
- **Personal Information**: Full Name, Email, Mobile, Designation, Role, Joined date, Status
- **Profile Picture**: Current picture display with upload/change/remove options
- **Security**: Change Password button

## Edit Profile
1. Click **Edit Profile** from the dropdown menu
2. Modify: First Name, Last Name, Email, Mobile Number
3. Click Save

## Change Password
1. Click **Change Password** from the dropdown menu
2. Enter: Current Password, New Password, Confirm New Password
3. Click Save

## Profile Picture
- **Format**: JPG, JPEG, PNG, WebP
- **Max size**: 5MB
- **Upload**: Click "Change Picture" and select a file
- **Remove**: Click "Remove Picture" to revert to initials avatar
