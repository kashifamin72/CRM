# Employee Management

Access: **Employees** in sidebar (Administrator only).

## Employee List
- **Search** by name, email, or designation
- **Filter** by Active/Inactive status
- Table shows: Avatar, Name + Email, Mobile, Designation, Role badges, Status, Created date
- Actions: Edit, Toggle Active

## Create Employee
1. Click **Add Employee**
2. Fill in: First Name, Last Name, Email, Mobile Number
3. Select **Designation** (Manager / Sales Officer)
4. Select **Role** (Manager / Sales Officer)
5. Set a **Password**
6. Click Save

## Edit Employee
1. Click ✏️ on any employee
2. Modify fields (password left blank to keep unchanged)
3. Click Save

## Toggle Active/Inactive
- Click the toggle button to deactivate/reactivate
- Inactive users cannot log in
- Their data (leads, follow-ups) is preserved
- Can be reactivated later

**Note:** The System Administrator account cannot be deleted or deactivated.
