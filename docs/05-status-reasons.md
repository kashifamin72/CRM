# Status Reasons (Close Reasons)

Access: **Status Reasons** in sidebar (Administrator only).

Manage the predefined reasons that appear when a user closes a lead as Won or Lost.

## Purpose
When a Sales Officer or Manager moves a lead to **Closed Won** or **Closed Lost**, they must:
1. Select a **Reason** from a dropdown (predefined list)
2. Enter a **Remark** (free text, minimum 10 characters)

This ensures every closed lead has a documented reason for audit and reporting.

## Default Reasons

### Closed Lost
1. Price too high
2. Chose competitor
3. No budget/timing
4. Product doesn't fit
5. Decision maker unavailable
6. Other

### Closed Won
1. Good price/value
2. Best solution fit
3. Strong relationship
4. Quick implementation
5. Good reviews/referral
6. Other

## CRUD Operations

### Add a Reason
1. Select **Closed Lost** or **Closed Won** tab
2. Click **Add Reason**
3. Enter the reason text
4. Click Save

### Edit a Reason
1. Click ✏️ on any reason
2. Modify the text
3. Toggle **Active** to enable/disable (inactive reasons won't appear in dropdown)
4. Click Save

### Delete a Reason
- Click 🗑️ to permanently remove
- Only possible if no leads use that reason
- Otherwise, deactivate instead

### Reorder
Drag the ⋮⋮ handle to reorder reasons (they appear in this order in the close modal).
