# Reports

Access: **Reports** in sidebar (Admin/Manager only).

## Filters
- **From Date / To Date** — Date range for lead creation
- **Sales Officer** — Filter by specific officer or "Unassigned"
- **Source** — Filter by lead source
- Click **Generate** to apply filters

## KPI Cards (8)
| Metric | Description |
|--------|-------------|
| Total | All leads in range |
| New | New leads |
| Contacted | Contacted leads |
| Qualified | Qualified leads |
| Proposal | Proposal stage leads |
| Won | Closed Won leads |
| Lost | Closed Lost leads |
| Est. Value | Sum of estimated values |

## Charts
- **Status Distribution** — Bar chart of leads by status
- **Conversion Breakdown** — Won/Lost/Active counts with win rate progress bar
- **Source Breakdown** — Doughnut chart + table with lead count and value by source

## Lead Details Table
Columns: Title, Customer, Status, Source, Value, Created By, Assigned To, Created Date
